# 什么是 Vibe Coding？

想象一下：

你跟 AI 说一句"帮我做个待办清单App，要有登录功能"，然后喝口咖啡，回来一看——代码写好了，运行正常，界面也挺好看的。

这就是 **Vibe Coding**。

---

## 传统 Coding vs Vibe Coding

| | 传统 Coding | Vibe Coding |
|---|---|---|
| **写代码** | 程序员手敲每一行 | 描述需求，AI 帮你写 |
| **学习曲线** | 需要学编程语言、框架、工具 | 会说话就能做 |
| **速度** | 慢，需求→写代码→调试→改bug | 快，说清楚就行 |
| **门槛** | 高，没学过编程很难上手 | 低，拥抱自然语言 |

**Vibe Coding 不是"不用写代码"那么简单。**

它的核心是：**用自然语言驱动 AI，让 AI 帮你完成编码工作，而你专注在"想要什么"这件事上。**

---

## 为什么这是未来？

1. **想法 → 实现的距离大幅缩短**
   以前：有个App想法 → 学编程 → 几个月 → 上线
   现在：有个App想法 → 告诉AI → 几天甚至几小时 → 上线

2. **AI 比想象中更懂代码**
   现代 AI 已经能写出高质量代码，还能帮你调试、解释、优化。

3. **创意不再受技术限制**
   不是所有人都学过编程，但每个人都有好点子。Vibe Coding 让创意不再因为"我不会写代码"而被埋没。

---

## 如何做 Vibe Coding？

**只需要 4 步：**

1. **说清楚你要什么** — 用自然语言描述你的需求，越具体越好
2. **让 AI 写代码** — 把需求丢给 AI，让它生成代码
3. **检查和调整** — 看看效果，不满意就说"这里改一下"
4. **迭代优化** — 不断微调，直到达到你想要的结果
