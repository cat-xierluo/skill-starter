# Issue: 概念入门模块 - 新手必读

## 背景

skills-starter 需要成为真正的新手友好项目，让完全不懂 Skill、Vibe Coding、GitHub 的人也能通过阅读项目文档 + 与 AI 对话来入门。

## 任务描述

在项目根目录创建 `concepts/` 目录，新增以下文档：

### 1. `concepts/WHAT-IS-SKILL.md`
- 什么是 Skill？（用类比的方式解释，让小白理解）
- Skill 和 Plugin、Extension 的区别
- 为什么 Skill 很重要？
- Skill 能做什么？
- 示例：生活中的 Skill（比如手机的"相机"Skill、"地图"Skill）

### 2. `concepts/WHAT-IS-VIBE-CODING.md`
- 什么是 Vibe Coding？（不用写代码，用自然语言和 AI 协作）
- Vibe Coding vs 传统 Coding
- 为什么这是未来？
- 如何做 Vibe Coding？（步骤）
- 本项目就是 Vibe Coding 的 starter

### 3. `concepts/WHAT-IS-GITHUB.md`
- 什么是 GitHub？（不是"gayhub"，是程序员的"云硬盘+协作平台"）
- 类比：GitHub ≈ 程序员的朋友圈/云盘/版本控制
- 为什么我们需要 GitHub？
- 基本概念：Repository, Commit, Branch, Pull Request

### 4. `concepts/GLOSSARY.md`
- 常用术语表（用简单语言解释）
- Agent, CLI, SDK, API, Token, SSH, Clone, Push, Pull, Fork, Star...
- 每个术语：英文名 + 中文翻译 + 30字内的解释 + 可选示例
